A premium React web application demonstrating fundamental cryptographic concepts, including Caesar and Vigenère ciphers. Developed for **DecodeLabs Cyber Security Project 2**.

## ✨ Features

- **Caesar Cipher:** Classic substitution cipher with customizable numerical shifts.
- **Vigenère Cipher:** Advanced polyalphabetic substitution cipher utilizing string keywords.
- **Real-Time Processing:** Encrypts and decrypts text instantly as you type.
- **Premium UI/UX:** Dark mode aesthetic with glassmorphism, smooth animations, and neon glowing accents tailored for a "Cyber Security" theme.
- **Responsive Design:** Fully functional across desktop and mobile devices.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/thepriyanshbhatt/decodelabs-project-2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd decodelabs-project-2
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173/`.
